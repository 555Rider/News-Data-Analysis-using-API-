# News-Data-Analysis-using-API-
## Project Overview
This project focuses on collecting and analyzing news data using APIs. The goal is to understand trends and keywordin news topics by transforming raw data into meaningful insights.

## Objective
Retrieve structured news data from online sources
Clean and organize raw data for analysis
Identify trends and pattern in news content

## Tools & Technologies
Python
Requests (API calls)
Pandas
Jupyter Notebook

## STEP OF WORKFLOW

Data Collection Process-
Used API to retrieve news data in JSON format
Extracted relevant fields such as title, date, and category
Stored and structured the data into a tabular format

Data Cleaning & Preparation-
Removed missing(none) or incomplete records
Converted date formats into usable structure
Organized data into Pandas DataFrame for analysis

Data Analysis-
Performed exploratory data analysis (EDA)
Analyzed frequency of keyword and topics across different sources
Compared similar keyword across multiple news platforms
Identified trending topics over time

## Key Insights
Certain topics appear more frequently during specific periods
News trends can reflect changes in public interest over time

## What I Learned From This Project
How to retrieve structured data using APIs
How to process JSON data into analysis-ready format
The importance of data cleaning before analysis
Basic workflow of a data analysis pipeline
